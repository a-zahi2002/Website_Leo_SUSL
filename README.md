# Leo Club Website

A modern, responsive website for the Leo Club built with React, Tailwind CSS, and Framer Motion.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

## Customization

-   **Content**: Edit `src/data.ts` to update text, images, projects, and leadership members.
-   **Styling**: Tailwind CSS is used for styling. Check `src/index.css` for global styles and theme variables.
-   **Images**: Replace image URLs in `data.ts` with your own images.

## Features

-   **Hero Section**: Impactful welcome with CTA.
-   **About**: Mission, Vision, Values.
-   **Projects**: Filterable grid (Completed, Ongoing, Upcoming).
-   **Leadership**: Executive and Board members.
-   **Gallery**: Masonry layout of events.
-   **Contact**: Inquiry form and social links.
